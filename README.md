# Swin-Transformer-Based-KHCR-
A  Swin Transformer based  approach for Robust  Kannada Handwritten Character Recognition   with Progressive Segmentation and Feedback-driven refinement
Kannada handwritten OCR is challenging due to complex ligatures, high intra-class variation, and the lack of standardized datasets, making robust recognition difficult for traditional methods like CNNs and neural networks.
The Swin Transformer provides a powerful solution by using a hierarchical structure and shifted window attention. It captures both local details and global context from images. 
We propose an end-to-end framework with centroid refinement, progressive segmentation, and feedback-storage for future learning <img width="3534" height="216" alt="image" src="https://github.com/user-attachments/assets/d31ae198-65c3-45a1-a416-fceec5c9760c" />

Data quality emerged as the primary driver of performance, with significant improvement achieved through careful curation. 
The Swin Transformer effectively captures both local stroke details and global ligature structures
.Kannada characters involve loops, conjuncts, and diacritics, requiring context-aware modeling.
 Centroid-based refinement improves prediction consistency and reduces confusion. 
Progressive segmentation enables efficient word-level recognition without sequence models.
